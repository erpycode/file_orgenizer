# 📁 مرتب سازی فایل ها 

> **فایل‌های پراکنده‌ات رو در چند ثانیه مرتب و دسته‌بندی کن — بدون دردسر!**

🌐 **زبان‌ها:** [English](README.md) · [فارسی](README.fa.md)

---

## ✨ این چیه؟

**File Organizer** یه اسکریپت سبک Python‌ه که پوشه‌ات رو اسکن می‌کنه و فایل‌ها رو به‌صورت خودکار داخل زیرپوشه‌های دسته‌بندی‌شده می‌ریزه. دیگه خداحافظ پوشه‌های به‌هم‌ریخته! 👋

---

## 📂 دسته‌بندی‌های پشتیبانی‌شده

| دسته‌بندی | پسوندها |
|---|---|
| 🖼️ تصاویر | `.jpg` `.jpeg` `.png` `.gif` `.bmp` `.tiff` `.svg` `.webp` |
| 📄 اسناد | `.pdf` `.doc` `.docx` `.txt` `.xls` `.xlsx` `.ppt` `.pptx` |
| 🎬 ویدیو | `.mp4` `.mkv` `.avi` `.mov` `.wmv` |
| 🎵 صوت | `.mp3` `.wav` `.aac` `.flac` `.ogg` |
| 🗜️ آرشیو | `.zip` `.rar` `.tar` `.gz` `.7z` |

## 🚀 شروع کار

### پیش‌نیازها

- Python نسخه **3.6** به بالا
- بدون هیچ پکیج خارجی — فقط کتابخونه‌های استاندارد!

### نصب

```bash
# کلون یا دانلود کن
git clone https://github.com/yourname/file-organizer.git
cd file-organizer

# اسکریپت رو داخل پوشه‌ای که می‌خوای مرتب کنی بذار
```

### اجرا

```bash
python file_organizer.py
```

## 🎮 نحوه کار

وقتی برنامه رو اجرا می‌کنی، یه منوی تعاملی می‌بینی:

```
********************

📁 File Organizer

********************

1.🟡 Move File To Folders
2.🔵 Copy Files To Folders
3.🟢 Dry Mode
4.🔴 Exit
```

### گزینه‌های منو

#### 🟡 گزینه `1` — جابه‌جایی فایل‌ها
فایل‌های شناخته‌شده رو از پوشه مبدأ **منتقل** می‌کنه داخل زیرپوشه مربوطه در `sorted/`. فایل اصلی از محل قبلی‌اش **حذف می‌شه**.

```
🟡 photo.jpg 🟡 Moved to images
🟡 report.pdf 🟡 Moved to documents
```

#### 🔵 گزینه `2` — کپی فایل‌ها
فایل‌های شناخته‌شده رو **کپی** می‌کنه داخل زیرپوشه مربوطه در `sorted/`. فایل اصلی **سر جاش می‌مونه** — ایده‌آل برای وقتی که نمی‌خوای چیزی حذف بشه.

```
🔵 song.mp3 🔵 copy to audio
🔵 archive.zip 🔵 copy to archives
```

#### 🟢 گزینه `3` — حالت شبیه‌سازی (Dry Mode)
بدون اینکه **هیچ فایلی لمس بشه**، نشون می‌ده که کدوم فایل کجا می‌ره. قبل از هر کاری اول این گزینه رو بزن!

```
🟢 video.mp4 🟢 will be copy / move to videos
🟢 notes.txt 🟢 will be copy / move to documents
```

#### 🔴 گزینه `4` — خروج
از برنامه خارج می‌شه. همین! 😄

## 🗂️ ساختار خروجی

بعد از اجرا، پوشه `sorted/` اینطوری به نظر می‌رسه:

```
sorted/
├── 🖼️  images/
├── 📄  documents/
├── 🎬  videos/
├── 🎵  audio/
└── 🗜️  archives/
```

---

## ⚙️ شخصی‌سازی

می‌خوای دسته‌بندی یا پسوند جدید اضافه کنی؟ فایل `file_organizer.py` رو باز کن و دیکشنری `FILE_CATEGORIES` رو ویرایش کن:

```python
FILE_CATEGORIES = {
    "images": [".jpg", ".jpeg", ".png", ...],
    "code": [".py", ".js", ".ts", ".html"],   # ← دسته‌بندی دلخواه اضافه کن!
    ...
}
```

می‌خوای نام پوشه خروجی رو عوض کنی؟ `target_path` رو تغییر بده:

```python
target_path = base_path / "sorted"   # ← "sorted" رو به هر چیزی تغییر بده
```

## 📋 نکات مهم

- فایل‌هایی با **پسوند ناشناخته** دست‌نخورده باقی می‌مونن.
- پوشه `sorted/` و تمام زیرپوشه‌ها **به‌صورت خودکار** ساخته می‌شن.
- اجرای برنامه از داخل پوشه `sorted/` توصیه نمی‌شه.
