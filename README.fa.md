# 📁 مرتب سازی فایل ها 

> **فایل‌های پراکنده‌ات رو در چند ثانیه مرتب و دسته‌بندی کن — بدون دردسر!**

🌐 **زبان‌ها:** [English](README.md) · [فارسی](README.fa.md)

---

## ✨ این چیه؟

**File Organizer** ابزاری سبک Python‌ه که فایل‌ها رو به‌صورت خودکار داخل زیرپوشه‌های دسته‌بندی‌شده می‌ریزه. دو نسخه داره:

- **حالت خط فرمان** (`file_organizer.py`) — منوی تعاملی در ترمینال
- **حالت گرافیکی** (`file_organizer_gui.py`) — رابط گرافیکی مدرن با کنترل‌های بصری

---

## 📂 دسته‌بندی‌های پشتیبانی‌شده

| دسته‌بندی | پسوندها |
|---|---|
| 🖼️ تصاویر | `.jpg` `.jpeg` `.png` `.gif` `.bmp` `.tiff` `.svg` `.webp` |
| 📄 اسناد | `.pdf` `.doc` `.docx` `.txt` `.xls` `.xlsx` `.ppt` `.pptx` |
| 🎬 ویدیو | `.mp4` `.mkv` `.avi` `.mov` `.wmv` |
| 🎵 صوت | `.mp3` `.wav` `.aac` `.flac` `.ogg` |
| 🗜️ آرشیو | `.zip` `.rar` `.tar` `.gz` `.7z` |

**در حالت گرافیکی می‌تونید دسته‌بندی‌های دلخواه اضافه کنید!**

## 🚀 شروع کار

### پیش‌نیازها

- Python نسخه **3.6** به بالا
- برای حالت گرافیکی: `python3-tk` (معمولاً پیش‌نصب است)

### نصب

```bash
git clone https://github.com/yourname/file-organizer.git
cd file-organizer
```

### اجرای نسخه خط فرمان

```bash
python file_organizer.py
```

### اجرای نسخه گرافیکی

```bash
python file_organizer_gui.py
```

## 🖥️ ویژگی‌های حالت گرافیکی

### رابط کاربری مدرن
- تم تیره با رنگ بنفش
- چیدمان واکنش‌گرا با فاصله‌گذاری مناسب

### انتخاب پوشه مبدأ
- **دکمه Browse** — انتخاب هر پوشه‌ای
- **دکمه Current Folder** — استفاده از پوشه‌ای که برنامه در آن قرار دارد

### حالت‌های عملیاتی
- **Move** — فایل‌ها به پوشه exported منتقل می‌شوند (اصلی حذف می‌شوند)
- **Copy** — فایل‌ها در پوشه exported کپی می‌شوند (اصلی سر جا می‌مانند)

### دسته‌بندی دلخواه
دسته‌بندی‌های دلخواه خودتون رو اضافه کنید:
1. نام پوشه رو وارد کنید (مثلاً `projects`)
2. پسوندهای موردنظر رو وارد کنید (مثلاً `.py,.js,.html`)
3. روی **Add** کلیک کنید
4. فایل‌هایی که این پسوندها رو دارند به آن پوشه ریخته می‌شن

### حالت شبیه‌سازی (Dry Run)
پیش از انجام کار، می‌بینید کدوم فایل‌ها کجا می‌رن بدون اینکه فایلی تغییر کنه.

## 🎮 حالت خط فرمان

منوی تعاملی ترمینال با گزینه‌های:
- `1` — جابه‌جایی فایل‌ها
- `2` — کپی فایل‌ها  
- `3` — حالت شبیه‌سازی (فقط پیش‌نمایش)
- `4` — خروج

---

## 🗂️ ساختار خروجی

فایل‌ها در این ساختار ذخیره می‌شن:
```
[پوشه-انتخابی]/
└── exported/
    ├── 🖼️  images/
    ├── 📄  documents/
    ├── 🎬  videos/
    ├── 🎵  audio/
    ├── 🗜️  archives/
    └── [پوشه‌های دلخواه‌تان]/
```

---

## ⚙️ شخصی‌سازی

فایل `FILE_CATEGORIES` رو در کد اصلاح کنید:

```python
FILE_CATEGORIES = {
    "images": [".jpg", ".jpeg", ".png", ...],
    "code": [".py", ".js", ".ts", ".html"],  # دسته‌بندی دلخواه اضافه کنید!
    ...
}
```

---

## 📋 نکات مهم

- فایل‌هایی با **پسوند ناشناخته** دست‌نخورده باقی می‌مونن
- پوشه `exported/` به‌صورت خودکار ساخته می‌شه
- اجرای برنامه از داخل پوشه `exported/` توصیه نمی‌شه
- برای حالت گرافیکی روی اوبونتو/دبین `sudo apt install python3-tk`
